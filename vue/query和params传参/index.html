Query和params传参

vue-router query和params传参(接受参数)的区别

  • query方式传参和接受参数
//传参
this.$router.push({
  path: '/login',
  query: {
    id: this.id
  }
})

//接受参数
this.$route.query.id

==传参是this.$router，是一个路由。==

==接收参数this.$route，是一个对象。==

  • params方式传参和接受参数
//传参: 
this.$router.push({
        name:'xxx'
        params:{
          id:id
        }
      })

//接收参数:
this.$route.params.id

注意：params传参，push里面只能是name命名的组件名name:'xxx',不能是path:'xxx'，因为params只能用name来引入路由，如果这里写成了path，接收参数页面会是undefined！！！

重点：

直白来讲query相当于get请求，页面跳转时会在地址栏中看到请求的参数，而params相当于post请求，参数不会在地址栏中显示
