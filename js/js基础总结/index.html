javaScript是什么

javaScript是一种运行在客户端（浏览器）的脚本语言

脚本语言：不需要编译，读取一句 解析一句 一句报错 下一句就不会执行；
执行过程：源代码 - 预解析 - 运行；
编译语句：需要编译，编译一旦出错 整个程序都不会运行
执行过程：源代码-编译后的字节码文件-运行

函数

函数的定义：一堆特定代码的集合体，负责完成某项特定任务；相对于其他代码而言，具有相对独立性，一般会有参数和返回值

函数就是一个功能，具备复用性

声明函数和调用函数

//声明函数
function 函数名(形参){
    //函数体
};

//调用函数
函数名(实参)

说明：

  • 函数不会自己主动执行，必须要调用之后之后才会执行
  • 函数的命名尽量有意义，遵守动词或者动词+名词这样的方式
  • 函数一次声明可以多次调用，每一次调用都会执行一次函数体
  • 函数内部还可以调用其他函数

函数参数：

  • 形参和实参可以是多个
  • 当形参和实参的长度不相等时；多出来的形参是undefined,多出来的实参可以忽略，也可以用arguments（实参列表）来接收，是个伪数组

函数返回值

​ 就是函数体里return后面的值，

  • 函数的返回值不是必要的，在函数执行完毕之后产生一个结果，这个结果要用到其他地方时就需要返回值；如果当前函数没有返回值浏览器就会默认返回一个undefined

函数细节补充

  1. 两个相同的函数，后面的就会覆盖前面的
  2. 实参和形参的个数可以不一样
  3. 函数碰到return就不会向下执行了，相当于函数结束了。循环的终止用break。
  4. 在函数里return相当于return undefined.

作用域

作用域:变量起作用的区域

全局作用域：函数外面的区域

局部作用域：函数内部的区域

  • 全局变量：在全局作用域var的变量，可以在任何区域进行读取和修改
  • 局部变量：在局部作用域var的变量，只能在当前函数内部读取和修改

变量访问原则：在能够访问的情况下，先局部，后全局。==> 就近原则

创建对象

四种方式：

第一种：字面量形式

var obj = {
	name:'hh',
	age:18
}

第二种方式：构造函数

var obj = new Object();
obj.name = 'hh';
obj.age = 18;

第三种方式：工厂函数批量创建函数

function creatPerson(name,age){
	var obj = {
		name:name,
		age:age
	};
	return obj;
}
creatPerson('hh',18)

优点：可以同时创建多个对象

缺点：麻烦，没有原型关系，是object类型，范围太宽广

第四种：自定义构造函数

//构造函数名大写
function Person(name,age){
	this.name = name;
	this.age = age;
}
var hh = new Person('hh',20);

new运算符

new构造函数执行四步：

  • 执行构造函数
  • 返回这个函数

new运算符一般配合构造函数使用，这种方法创建对象更加简单。并且有原型关系，可以看到数据类型

//参数
//1、不传递参数 ==> 默认会用逗号进行连接
//2、传第参数 ==> 数组的每一项用传递的参数这个符号进行连接

  • 数组的翻转和排序
array.reverse(); //反转数组，直接操作原数组
array.sort(); //数组的排序，默认按照字母进行排序，修改的是原数组

//sort方法可以传递一个函数作为参数，这个参数用来控制数组如何进行排序
arr.sort(function(a,b){
    //如果返回值大于0，则交换位置
    return a - b;  //升序
    return b - a;   //降序
})
  • 数组查找元素
//indexOf方法用来查找数组中某个元素第一次出现的位置，如果找不到，就返回-1;
array.indexOf(search,[fromIndex]);

//lastIndexOf()从后面开始查找数组中元素出现的位置，如果找不到，就返回-1
//注意从后往前找还是返回的元素出现的位置还是返回元素在数组中的正常的下标
array.lastIndexOf(search,[fromIndex]);

// 参数: 1. 需要检测的元素  2. 操作的起始下标
  • 数组的拼接于提取
//concat:数组合并，不会影响原来的数组，会返回一个新的数组。
var newArr = array.concat(array2);

//slice:数组切分，复制数组的一部分到一个新的数组，并返回这个数组，不修改原数组
var newArray = array.slice(begin, end);

//参数：
//1、不传递参数：==> 直接复制一份一样的数组
//2、只传递一个参数  ==> 操作的下标 ==> 直接从这个下标的数据一直截取到最后一个数据
//3、传递两个参数 ==> 开始下标，结束下标 ==> 从第一个参数的下标截取到第二个参数下标的之间的数据，但是不包括最后一个参数下标的数据

 1、array = [];  //数组直接赋值为一个空数组，常用
